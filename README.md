Nxt Trendz (E-Commerce Application) Developed a responsive E-Commerce web application with secure authentication, product browsing, and shopping cart functionality.

Built a responsive E-Commerce web application using React.js with reusable components and dynamic routing. Implemented JWT-based authentication and protected routes to ensure secure user access to product and cart pages. Developed login and authentication flow with session management using Cookies. Integrated REST APIs to fetch and display product details dynamically. Implemented product search, sorting, and filtering functionality for an improved shopping experience. Built a dynamic shopping cart system with add/remove items, quantity management, and real-time total price calculation. Used React Context API for efficient global state management across the application. Designed a responsive UI/UX using HTML, CSS, Flexbox, and media queries for seamless mobile and desktop experience. Implemented conditional rendering, loading states, and error handling to improve usability and application performance. Developed product details and similar products sections for enhanced product exploration.
